# Guide utilisateur — PotagerConnect

## Je suis jardinier

### Créer mon compte
1. Ouvrir l'application, onglet **Inscription**
2. Renseigner prénom, nom, email, mot de passe (8 caractères minimum)
3. Valider — vous êtes connecté automatiquement

### Obtenir une parcelle
Les parcelles sont attribuées par le responsable du jardin. Tant qu'aucune parcelle ne vous est attribuée, l'écran « Ma parcelle » vous invite à le contacter.

### Déclarer une culture
1. Écran **Ma parcelle**
2. Bouton **+ Ajouter**
3. Saisir la plante (tomate, radis, courgette...) et la date de plantation
4. Valider

La date de récolte prévue est calculée automatiquement si la plante est connue.

### Suivre mes cultures
Chaque culture affiche son statut :
- 🟡 **En cours** — elle pousse
- 🟢 **Récoltée** — c'est fait

Bouton **Récolter** pour passer une culture en « récoltée ».

### Suggestions de plantation
L'encart vert « À planter ce mois-ci » propose les plantes de saison. Les familles déjà cultivées sur votre parcelle cette année en sont exclues : c'est la **rotation des cultures**, elle préserve le sol.

## Je suis responsable du jardin

### Voir la carte des parcelles
Écran d'accueil : toutes les parcelles, avec un code couleur.
- 🟩 **Vert** — parcelle libre
- ⬜ **Blanc** — parcelle attribuée, avec le nom du jardinier

### Attribuer une parcelle
1. Cliquer une parcelle libre
2. Choisir un jardinier dans la liste
3. **Attribuer**

Une parcelle déjà attribuée ne peut pas être réattribuée directement : il faut d'abord la libérer.

---

## Questions fréquentes

**J'ai oublié mon mot de passe.** Contactez le responsable du jardin.

**Puis-je voir la parcelle d'un autre jardinier ?** Non. Chaque jardinier n'accède qu'à la sienne.

**Ma plante n'est pas reconnue.** Elle est acceptée, mais sans calcul automatique de date de récolte.
